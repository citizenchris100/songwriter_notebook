🎛 Overdub latency spike

Throwaway diagnostic for the 4-track tape deck. It measures the monitoring round-trip (app output → EVO → back into an input) so overdubs can be time-aligned. Run it on the same device + interface you record with (iPad + Audient EVO 4), installed / offline too.

  1. Loop the output back to input 1. Best: a short cable from the EVO's line/headphone out to input 1. No cable? Put input 1's mic right against the headphone/speaker and turn it up — the click just has to come back in clearly.
  2. Turn input 1's gain up enough to see the returning click, but avoid clipping.
  3. Tap Measure a few times and read the median. Enter that value (seconds) as the deck's monitorLatencySec for this device.
